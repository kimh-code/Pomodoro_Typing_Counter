# Pomodoro Typing Counter

A productivity tool combining a Pomodoro timer with a typing counter.
Built with PyQt5.

## Features
- Counts typed characters in real time
- Pomodoro timer with focus and rest modes
- **Save and borrow rest time** — extra focus time beyond your target gets 
  saved as rest time. You can borrow that saved time for longer breaks.
- Window opacity changes based on session progress
- Customizable focus time, rest time, and typing goals

## Requirements
```bash
pip install PyQt5 pynput
```

## How to use
```bash
python Pomodoro_Typing_Counter7.py
```

## Files
- `Pomodoro_Typing_Counter7.py` — latest version (recommended)
- `Pomodoro_Typing_Counter4.py` — earlier version
- `typing_counter.py` — standalone typing counter
